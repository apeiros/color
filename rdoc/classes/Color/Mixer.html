Class Color::Mixer
In: lib/color/mixer.rb
Parent: Object

this class is intended to make mixing colors simple i.e. you can alter single attributes of a color, even attributes that belong to another model (Mixer will convert forth and back for the operation). Example: 

  red = Color.rgb(255,0,0)
  mix = red.to_mixer
  mix.hue += 1.0/3
  mix.color # => RGB: 0, 255, 0, 0
  mix.alpha += 128
  mix.color # => RGB: 0, 255, 0, 128
